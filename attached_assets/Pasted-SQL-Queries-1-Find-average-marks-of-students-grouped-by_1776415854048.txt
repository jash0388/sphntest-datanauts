SQL Queries
1. Find average marks of students grouped by Name.
Answer: SELECT Name, AVG(Marks) FROM Students GROUP BY Name;
2. Count number of students with same marks.
Answer: SELECT Marks, COUNT(*) FROM Students GROUP BY Marks;
4. Show names having average marks > 70.
Answer: SELECT Name, AVG(Marks) FROM Students GROUP BY Name HAVING AVG(Marks) >
70;
5. Find marks that appear more than once.
Answer: SELECT Marks, COUNT(*) FROM Students GROUP BY Marks HAVING COUNT(*) > 1;
Create a view for students with marks > 75.
Answer: CREATE VIEW HighScorers AS SELECT ID, Name, Marks FROM Students WHERE
Marks > 75;
Display the created view.
Answer: SELECT * FROM HighScorers;
